# PetCarousel Component

## 设计概述

PetCarousel 组件采用了与设计指南一致的上下两部分卡片布局：

### 卡片结构

```
┌─────────────────────────────────────────────┐
│  🐕  Charlie          │  每日目标           │
│      Golden Ret...    │  6/8                │
│      3岁              │  [████████░░]       │
├─────────────────────────────────────────────┤
│  📝  下个任务                                │
│      下午散步时间                            │
│      🕐 14:00                                │
└─────────────────────────────────────────────┘
```

### 上半部分（Top Section）
- **左侧**：宠物信息
  - 64x64 emoji头像（圆角方形）
  - 宠物名字（18px，粗体）
  - 品种（14px）
  - 年龄（12px）
  - 主宠徽章（如果是主宠）

- **右侧**：每日目标
  - "每日目标"标签
  - 完成数/总数（6/8）
  - 进度条

### 下半部分（Bottom Section）
- **下一个任务**
  - 图标（32x32容器）
  - 任务标题
  - 时间信息

## 动画效果

轮播支持以下动画：
- **Scale**: 中心卡片 1.0，两侧 0.88
- **Opacity**: 中心卡片 1.0，两侧 0.6
- **TranslateY**: 两侧卡片向下偏移 20px

## Mock数据

当前使用的mock数据：
- goalsCompleted: 6
- goalsTotal: 8
- nextTask: { title: '下午散步时间', time: '14:00', icon: 'walk' }

## Props

- `pets`: 宠物数组
- `loading`: 加载状态
- `currentPetIndex`: 当前选中的宠物索引
- `onPetChange`: 宠物切换回调
- `onAddPet`: 添加宠物回调
- `isDarkMode`: 深色模式状态

## 样式特点

- 橙色主题 (#F97316)
- 半透明背景效果
- 毛玻璃效果（backdrop-filter）
- 响应式卡片大小
- 支持深色模式
